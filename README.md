# Установка и настройка форума bbPress для сайта студии йоги «Шанти»

---

## 1. Установка плагина bbPress

### Шаги
1. В админке WordPress перейти: **Плагины → Добавить новый**
2. В строке поиска ввести `bbPress`
3. Найти плагин **bbPress** (автор: The bbPress Community)
4. Нажать **Установить** → **Активировать**

### Результат
После активации в левом меню появился раздел форумы


## 2. Создание структуры форума

### Шаги
1. Перейти: **Форумы → Новый форум**
2. Создать главный форум:
   - **Название:** Форум студии «Шанти»
   - **Описание:** Пространство для общения учеников и преподавателей. Здесь вы можете задать вопрос, поделиться опытом, обсудить практики и просто познакомиться с единомышленниками.
   - **Тип:** Форум
   - Нажать **Опубликовать**

3. Создать три раздела (подфорума):
   - Перейти: **Форумы → Новый форум**
   - Для каждого указать **Родительский форум:** Форум студии «Шанти»
   - **Тип:** Форум

## 4. Настройка ролей

### Шаги
1. Перейти: **Пользователи → Все пользователи**
2. Выбрать пользователя → **Редактировать**
3. В блоке **Роль на форуме** выбрать: участник или модератор
4. Нажать **Обновить пользователя**

---

## 5. Создание тем

### Шаги
1. Перейти на сайт → Форум студии «Шанти»
2. Зайти в раздел (например, «Вопросы о практике»)
3. Нажать **Создать тему**
4. Заполнить и опубликовать

## 6. Добавление правил форума

### Шаги
1. Перейти: **Страницы → Добавить новую**
2. Название: **Правила форума**
3. Заполнить страницу
4. Опубликовать

### Правила включают
1. Уважение друг к другу (ахимса)
2. Открытость к разным точкам зрения
3. Запрет на медицинские советы
4. Запрет на спам
5. Конфиденциальность личных сообщений

## 7. Оформление форума в стиле сайта

### Шаги
1. Перейти: **Внешний вид → Настроить → Дополнительные стили**
2. Вставить CSS-код
3. Нажать **Опубликовать**

---

## 8. Добавление форума в меню сайта

### Шаги
1. Перейти: **Внешний вид → Меню**
2. В левом блоке «Форумы» отметить «Форум студии «Шанти»»
3. Нажать **Добавить в меню**
4. Перетащить пункт в нужное место
5. Нажать **Сохранить меню**
